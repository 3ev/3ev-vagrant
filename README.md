#3ev Basic Vagrant Box

This basic Vagrant box mirrors the setup on a few different servers.

##What's included?

* Ubuntu 12.04
* Apache 2.2
* MySQL 5.5 (root user/password: `root/root`)
* PHP 5.3, and some useful extensions
* Composer
* Memcache
* SQLite
* Node.js latest with RequireJS and Bower
* Ruby (OS version)
* Sphinx search
